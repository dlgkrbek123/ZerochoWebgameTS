typescript
- 자바스크립트의 슈퍼셋?
- 모든 기능을 담으면서 타입이라는 시스템을 추가
- 변수, 매개변수, return에 타입을 부여


- 언어이자 라이브러리
- 직접 실행되지 않고 js로 컴파일되어야 실행가능
- 컴파일 시점의 에러를 잡을 수 있다.

---------------------------------------------------------------------------

npm init -y
npm i typescript
sudo npm i -g typescript // 전역 설치

---------------------------------------------------------------------------

tsc 
- 타입스크립트 컴파일러
- 컴파일 에러코드로 에러를 유추가능

tsc gugudan.ts // js로 컴파일
tsc gugudan.ts -w // 파일을 워치하면서 재컴파일

---------------------------------------------------------------------------

- 일단 ts도 js랑 동일하게 작성가능
- 변수, 매개변수, return에 타입 부여
- 그렇게만 해도 명확한 에러는 찾아줘서 좋다.

---------------------------------------------------------------------------

- g옵션없이 npx로 명령어를 실행
- global 설치없이 라이브러리 실행
- 로컬 설치버전으로 명령어가 실행되어 좋다.

---------------------------------------------------------------------------

- package-lock.json에 정확한 라이브러리 버전이 명시되어있다.
- 서버에 올릴 때 같이 올리자
